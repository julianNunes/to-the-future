Gerenciamento - Orçamento
    - Select de ano
        - Ao fazer o change buscar todos os orçamentos do ano

    - Tabela com a lista de orçamentos do ano
        - Botão para editar/visualizar
        - Botao para remover Orçamento

    - Botão de criação do Orçamento
        - Abril Modal
            - Select de Mes (Obrigatorio)
            - CheckBox (default false) - perguntar se quer criar para o meses subquetenes ate o final do ano
            - Botao Salvar
            - Botao cancelar

    - Botão para "Clonar" um mes para outro
        - Check para: "Despesas Fixas", "Provisionamento" e "Metas"
